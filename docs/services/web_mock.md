# Web-Mock Service

> **Status:** Implemented (v0.2.0) · **Tier:** 1 · **Port:** 8001

Development UI shell for the Silvasonic Web Interface. Serves the complete UI
(FastAPI + Jinja2 + HTMX + Alpine.js + Tailwind CSS + DaisyUI) with hardcoded
mock data — no database required. Precursor to the production Web-Interface
(v0.8.0).
